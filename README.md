# Workshop: Lập đội ngũ AI Agent cho Sale & Marketing

Bộ tài liệu dạy hoàn chỉnh cho workshop online 2.5 giờ (phễu cho khóa live 6 buổi, CES Global).
Mục tiêu: bất kỳ giảng viên nào mở repo này cũng dạy được đúng chương trình.

- Thời gian mẫu: Thứ Bảy 11/07/2026, 20:00 - 22:30, online Zoom.
- Case demo xuyên suốt: thương hiệu mỹ phẩm thiên nhiên giả định Thảo An.

## Bắt đầu từ đâu
1. Giảng viên đọc `01-giang-vien/kich-ban-giang-day-chi-tiet.md` (kịch bản từng phút).
2. Xem `03-prompts/README.md` để biết prompt nào đi với file data nào.
3. Chuẩn bị theo `01-giang-vien/checklist-truoc-buoi.md`.
